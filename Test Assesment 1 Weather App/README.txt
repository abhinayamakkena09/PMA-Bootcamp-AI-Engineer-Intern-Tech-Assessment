Weather App
A simple web-based weather application that allows users to view current weather and a 5-day forecast by entering a city name or using their device's location.

How to Run
Download or clone the repository (if applicable).

Open the Weather-App.html file in any modern web browser (Chrome, Firefox, Edge, etc.).

Enter a city name in the input field and click Search, or click Use My Location to automatically fetch weather based on your geolocation.

Requires an internet connection to fetch data from the OpenWeatherMap API.

What This App Does
Fetches and displays current weather data including temperature, humidity, and wind speed.

Displays a 5-day weather forecast using data from OpenWeatherMap.

Allows users to search by:
City name
Device geolocation

Handles and displays errors like invalid city names or denied geolocation access.

API Used
OpenWeatherMap API
Current Weather Data
5 Day / 3 Hour Forecast
